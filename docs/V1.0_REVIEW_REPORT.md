## 二、详细评审

### P0 - 阻断级问题

#### 2.1 Hooks 未集成到 Claude Code

**当前状态**: ⚠️ 部分解决

**已实现部分**:
- `src/harnessgenj/harness/hooks_integration.py` - 完整的 Hooks 集成层
- `src/harnessgenj/engine.py` - 在关键方法中调用 Hooks：
  - `receive_request()` → validation + security hooks
  - `complete_task()` → post_task hooks
  - `develop()` → pre_task + post_task hooks
  - `fix_bug()` → pre_task + post_task hooks
  - `adversarial_develop()` → security hooks

**缺失部分**:
```json
// .claude/settings.json 当前内容
{
  "permissions": {
    "allow": [
      "WebFetch(domain:mp.weixin.qq.com)",
      "Bash(python -c ':*)"
    ]
  }
  // ❌ 缺少 hooks 配置
}
```

**影响分析**:
- Claude Code 的 Write/Edit 操作不会自动触发框架的审查机制
- CodeReviewer/BugHunter 无法自动介入
- GAN 对抗审查入口被阻断

**解决方案**:
在 `.claude/settings.json` 中添加 hooks 配置，连接到已有的 `harnessgenj_hook.py` 脚本。

#### 2.1 事件驱动角色触发

```python
# 新增 src/harnessgenj/harness/event_triggers.py

ROLE_TRIGGER_MATRIX = {
    "on_write_complete": ["code_reviewer"],
    "on_task_complete": ["bug_hunter", "tester"],
    "on_security_issue": ["code_reviewer", "bug_hunter"],
    "on_architecture_change": ["architect"],
}

class EventTrigger:
    def __init__(self, harness: Harness):
        self.harness = harness
    
    def trigger(self, event: str, context: dict):
        """触发对应角色"""
        roles = ROLE_TRIGGER_MATRIX.get(event, [])
        for role_type in roles:
            self.harness._collaboration.send_message(
                from_role="system",
                to_role=role_type,
                content=context
            )
```

#### 2.2 知识库结构化

# 增强 memory/manager.py

class KnowledgeEntry(BaseModel):
    id: str
    type: Literal["bug_fix", "decision_pattern", "architecture_change", "security_issue"]
    problem: str
    solution: str
    code_location: dict[str, Any] | None
    severity: str | None
    tags: list[str]
    created_at: datetime
    verified: bool = False

def store_structured_knowledge(self, entry: KnowledgeEntry):
    """存储结构化知识"""
    self._knowledge[entry.id] = entry.model_dump()
    # 同时建立索引
    self._build_indexes(entry)
